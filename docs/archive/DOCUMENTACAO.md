## 🔐 Acesso ao Sistema

### 🌐 URLs de Acesso

| Ambiente | URL | Status |
|----------|-----|--------|
| **Produção** | [vendacerta.up.railway.app](https://vendacerta.up.railway.app) | ✅ Ativo |
| **Local** | http://127.0.0.1:5000 | 🔧 Dev |

### 👤 Primeiro acesso (sem senha padrão)

- **Não existem credenciais/senhas padrão** no sistema.
- Defina `ADMIN_EMAIL` e `ADMIN_PASSWORD` e execute `python scripts/create_admin.py`.

## 🚀 Deploy Rápido

### Opção 1: Deploy Rápido (5 min)
📄 Siga o [DEPLOY_RAPIDO.md](DEPLOY_RAPIDO.md)

```bash
# 1️⃣ Acesse Railway e faça login com GitHub
# 2️⃣ Crie projeto do repositório cristiano-superacao/vendacerta
# 3️⃣ Adicione PostgreSQL database
# 4️⃣ Configure variáveis: SECRET_KEY, FLASK_ENV=production
# 5️⃣ Gere domínio: vendacerta.up.railway.app
# ✅ Deploy automático em 3-5 minutos!
```
